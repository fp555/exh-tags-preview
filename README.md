# E(x)-Hentai Tags Preview :panda_face::label::mag:
This is an unofficial userscript for e-hentai/exhentai (**NSFW sites** so no links) that displays related tags when you hover over a gallery link. If you can't be bothered opening galleries every time to see all tags and you don't want to use the native *Extended* layout, this is the script for you. This strives to be the smallest, most unintrusive script that could possibly do a decent enough job.

## Install
If your browser does not natively support userscripts, install something like [Tampermonkey](https://tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/get-it/), then go grab the [latest userscript release](https://github.com/fp555/exh-tags-preview/releases/latest/download/script.user.js).

## Configuration
Starting with v1.8.0 there is an options panel added to pages to configure the script behavior. (Since v1.6.0 the script can be configured by directly editing the stored values from your userscripts manager dashboard -- *not recommended anymore*).

Run the script once to let it setup its storage. [Check your release notes](https://github.com/fp555/exh-tags-preview/releases) for details on which options are available.

## Troubleshooting
### I found a bug! / It doesn't work!
- Stop using weird browsers that don't respect/implement basic web standards;
  - While I hear it may be possible to run userscripts in Internet Explorer, I expect mine not to work properly. I don't care about IE compatibility, and neither should you.
- Try using the latest version of Tampermonkey or Violentmonkey as your userscript manager. Anything else I don't support and you're on your own;
- This script uses emojis: [make sure](http://caniemoji.com/) your browser/OS combination can display them properly;
- The popup will be empty if a gallery has NO tags. Please be a useful member of the community and add some yourself;
- You are using E-Hentai and you have the "offensive content" warning enabled. To fix the empty popups you have to click "**Never warn me again**" on the warning page;
- You are probably trying to preview an inaccessible (permanently deleted, copyright claimed, banned in your country...) gallery. My script is not magic, and will never work for those;
- [Open an issue](https://github.com/fp555/exh-tags-preview/issues/new) and convince me it's my fault.
